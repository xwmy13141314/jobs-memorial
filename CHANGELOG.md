# 更新日志

## [1.0.5] - 2026-01-31

### 新增 ✨
- **AI 对话功能**:
  - 悬浮 AI 触发器（乔布斯形象图标，页面右下角）
  - 对话面板支持流式响应和打字机效果
  - 四个预设问题标签（产品取舍、审美纠偏、品牌灵魂、创新瓶颈）
  - 乔布斯人格化 AI 回复（犀利、直觉导向、完美主义风格）
- **浏览量统计系统**:
  - 全局 Footer 显示简洁访问量（图标+数字）
  - 关于页面添加详细统计卡片（总访问量、今日访问、最后更新时间）
  - API 接口支持 GET/POST 操作，数据持久化到 `data/visits.json`
  - 自动清理30天前的每日数据
  - 使用 sessionStorage 防止同会话重复计数

### 优化 🎨
- **手机端体验优化**:
  - 设计理念页面：文字颜色从 `text-gray-700` 改为 `text-gray-900`，提高对比度
  - 产品页面：左右切换箭头在手机端隐藏，底部添加上下切换按钮
  - AI 图标：桌面端保持中间右侧，手机端移到右下角避免遮挡内容
- **全局布局**: 在根布局添加 ChatWidget 组件，全站可用

### 技术改进 🔧
- 新增 `/api/chat` 路由处理 AI 对话请求
- 新增 `/api/visit` 路由处理浏览量统计
- 使用 Vercel AI SDK 实现流式响应
- 使用 Framer Motion 实现流畅动画效果

---

## 1. 项目概述

### 1.1 项目背景
创建一个以史蒂夫·乔布斯为主题的纪念网站，展示其生平、设计理念、伟大产品和经典语录，传递 "Think Different" 的精神。

### 1.2 核心目标
- **致敬传奇**：全面展示乔布斯的生平和成就
- **传递精神**：传播设计哲学和创新理念
- **教育资源**：为学习者和创业者提供灵感来源

---

## 2. 功能模块

### 2.1 首页 (Hero)
**功能描述**
- 全屏英雄展示区域
- 简洁有力的介绍文字
- "产品教父" 称号渐变色高亮
- "开始探索" 行动按钮

**技术实现**
- 响应式背景设计
- 向下滚动提示动画
- 平滑滚动导航

### 2.2 生平时间线
**功能描述**
- 可横向滚动的时间线卡片
- 按年份展示乔布斯的重要人生节点
- 统一卡片尺寸，保证视觉一致性

**内容结构**
- 年份标签
- 事件描述
- 相关标签（如 Apple、NeXT、Pixar）
- 详细说明按钮

### 2.3 设计理念
**功能描述**
- 10篇设计哲学文章
- 涵盖极简主义、专注、用户体验等核心主题
- 每篇文章独立页面展示

**文章列表**
1. 极简主义 (Simplicity)
2. 专注 (Focus)
3. 注重细节 (Attention to Detail)
4. 改变世界 (Change the World)
5. 设计驱动 (Design Driven)
6. 创新 (Innovation)
7. 完美主义 (Perfectionism)
8. 产品优先 (Product First)
9. 用户体验 (User Experience)
10. Think Different

### 2.4 伟大产品
**功能描述**
- 展示乔布斯推出的经典产品
- 左右切换浏览（桌面端）
- 上下切换浏览（移动端）
- 产品图片 + 简介展示

**产品列表**
- Apple II
- Macintosh
- iPod
- iPhone
- iPad
- 等经典产品

### 2.5 经典语录
**功能描述**
- 20条乔布斯经典名言
- 中英文对照展示
- 网格布局，响应式适配

**示例语录**
- "Stay hungry, stay foolish."
- "Innovation distinguishes between a leader and a follower."
- "People don't know what they want until you show it to them."
- 等...

### 2.6 学习资源
**功能描述**
- 推荐相关书籍
- 推荐纪录片和电影
- 外部链接展示

**资源类型**
- 书籍推荐（如《乔布斯传》）
- 纪录片推荐
- 演讲视频推荐

### 2.7 关于本站
**功能描述**
- 网站介绍和宗旨
- 技术栈说明
- 联系方式

### 2.8 全局导航
**功能描述**
- 顶部固定导航栏
- 所有页面统一显示
- 响应式菜单（移动端）

**导航项**
- 生平
- 设计
- 产品
- 语录
- 资源
- 关于

### 2.9 页脚
**功能描述**
- 版权信息
- 简洁的链接集合

## 3. 技术架构

### 3.1 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| **Next.js** | 16 | 前端框架 |
| **React** | 19 | UI 框架 |
| **TypeScript** | - | 类型系统 |
| **Tailwind CSS** | 4 | 样式框架 |
| **Framer Motion** | 12 | 动画库 |

### 3.2 目录结构

```
jobs memorial/
├── app/
│   ├── layout.tsx              # 根布局
│   ├── page.tsx                # 首页
│   ├── timeline/               # 生平时间线
│   ├── design/                 # 设计理念
│   ├── products/               # 伟大产品
│   ├── quotes/                 # 经典语录
│   ├── resources/              # 学习资源
│   └── about/                  # 关于本站
├── components/
│   ├── Hero.tsx                # 首页英雄区
│   ├── Navbar.tsx              # 导航栏
│   ├── Footer.tsx              # 页脚
│   ├── HighlightCards.tsx      # 亮点卡片
│   ├── MouseFollower.tsx       # 鼠标跟随效果
│   └── Analytics.tsx           # 分析组件
├── content/
│   └── design/                 # 设计理念文章
└── public/                     # 静态资源
```

### 3.3 设计规范

| 元素 | 规范 |
|------|------|
| **主色调** | 黑色 `#000000` + 灰色 `#F5F5F7` |
| **字体** | SF Pro / Inter (系统字体栈) |
| **圆角** | 12px - 24px |
| **阴影** | `0 4px 24px rgba(0,0,0,0.08)` |
| **动画** | 300ms ease-out |
